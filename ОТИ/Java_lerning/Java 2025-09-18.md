## Ввод-вывод информации Java
`С клавиатуры`

### Сложение двух чисел
```Java
import java.util.Scanner;

public class Main{
public static void main(String[] a){
	int a,b,c ;
	Scanner in=new Scanner(System.in);
	System.out.print("a="); a=in.nextInt();
	System.out.print("b="); b=in.nextInt();
	
	c = a+b;
System.out.print(c);
}}
```
